# Hertz Hunter

## はじめに

FPVドローン向けの低コスト [RF Explorer](https://j3.rf-explorer.com/) 代替品です。使用中の周波数の特定、バックグラウンドノイズの確認、映像送信機（VTX）の不具合診断などに役立ちます。低コスト（6,000円以下）で自作できるよう設計されています。

あるレースイベントで、破損したVTXが2つのチャンネルでフルパワーで送信し続け、別のパイロットの映像に干渉するという問題が発生しました。スペクトラムアナライザーがこの問題の診断に不可欠でした。破損したVTXだけを電源投入した際、スペクトルグラフに異なる周波数の2つのピークが確認できました。

本プロジェクトはこの便利なツールをパイロットやレース主催者が利用しやすくすることを目的としています。レース当日の道具袋に手軽に追加できます。一般的なRX5808映像受信機を使用して5645MHzから5945MHz（ローバンドチャンネルは5345MHzから5645MHz）をスキャンし、OLEDディスプレイにこの範囲の各周波数における受信信号強度（RSSI）のグラフを表示します。

## 機能

- スキャン画面から直接起動（素早いアクセスが可能）
- FPVレーシングドローンが映像に使用するRFスペクトラムのスキャン（5645MHzから5945MHz）および追加のローバンド（5345MHzから5645MHz）周波数
- VTXが送信している周波数を示すRSSIのグラフ表示
- メニュー操作と機器制御のための複数入力オプション
  - 3ボタン（`PREV`、`SEL`、`NEXT`）
  - ロータリーエンコーダー（反時計回りで`PREV`、中央クリックで`SEL`、時計回りで`NEXT`）
- 選択可能なスキャン間隔
  - 2.5MHz間隔は最高解像度（更新レートは最低）
    - RX5808は1MHzステップのみ対応のため、この間隔は最近傍整数に丸められます（例: 5800, 5803, 5805, 5808, ...）
  - 5MHz間隔は中程度の解像度と更新レート
  - 10MHz間隔は最低解像度（更新レートは最高）
- **ピークマーカー検出** — 完全スキャンパスが完了するたびに上位1〜2個のピーク周波数を自動検出し、グラフ上の該当位置に下向き三角インジケーターでマーキング
- **マーカー数の設定** — 設定メニューでマーカーの数（オフ、1個、2個）を設定可能
- **マーカー表示の切り替え** — `PREV`と`NEXT`を同時押しでスキャン画面のマーカー表示をオン/オフ切り替え
- バッテリー電圧のモニタリング（低電圧アラーム付き）。メインメニューとスキャン画面の両方に表示
- 既知の低/高RSSI値間のキャリブレーション
- 選択した周波数のキャリブレーション済み信号強度の表示
- 再起動後も設定を保持
- 他のソフトウェアとの連携方法が複数
  - Wi-FiホットスポットからアクセスできるAPI
  - クライアントプログラムとのUSBシリアル通信（[公式クライアント](https://github.com/odddollar/Hertz-Hunter-USB-client)）
